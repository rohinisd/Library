# Cursor skills (PaaS template)

| Skill | Use when |
|-------|----------|
| **create-app-and-deploy** | New MVP from this template + deploy (any topic). Start here for “create an app and deploy it”. |
| **workflow-deploy-vercel-git** | Live site shows wrong/old code; Vercel ↔ Git; team URL vs `project.vercel.app`. |
| **mvp-paas-auth-oauth** | JWT, bcrypt, Google OAuth, `FRONTEND_URL`, Render OAuth env. |
| **monitor-logs-and-fix** | 500s, “server not configured”, post-deploy debugging. |

Each folder contains `SKILL.md` with frontmatter (`name`, `description`) for Cursor discovery.
